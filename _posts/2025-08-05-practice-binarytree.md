# LeetCode94 【二叉树的中序遍历】

## 题目描述

给定一个二叉树的根节点root，返回它的中序遍历。

## 输入输出样例 #1

### 输入 #1

```
root=[1,null,2,3]
```

### 输出 #1

```
[1,3,2]
```

### 输入 #2

```
root=[]
```

### 输出 #2

```
[]
```

### 输入 #3

```
root=[1]
```

### 输出 #3

```
[1]
```

## 说明/提示

- 树中节点数目在范围[0, 100]内
- 节点绝对值<=100
